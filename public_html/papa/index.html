Cascading Style Sheets (css 00)

Cascading Style Sheets (CSS) es la herramienta que diseñadores web y desarrolladores utilizan al lado de los lenguajes de marcas tales como HTML y XHTML para construir web. CSS proporciona a las navegadores la información necesaria para controlar el aspecto visual de la página web. Tal como la posición de los elementos HTML, fondos, colores e imágenes, y mucho mas.

Las técnicas avanzadas de CSS proporcionan a los autores de sitios web la habilidad de diseños a la medida para navegadores web de teléfonos móviles, , tan buenos como los estructuras que se necesita para crear sitios web para navegadores desktop. La introducción será para escribir CCS par dispositivos moviles.

El uso más comun de CSS es para dar estilo a las páginas web, y en combinación con HTML o XHTML (utilizados para mostrar el contenido) y con JavaScript (el cual se utiliza par adicionar interactividad al sitio), CSS es una poderosa herramienta.
